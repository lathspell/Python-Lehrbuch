# 2.3 Logische Operatoren und Verschachtelung

Bedingungen kombinieren und verschachteln

---

## 🎯 Problem

Du hast gelernt, wie man einfache Bedingungen mit `if` prüft. Aber was, wenn du:

- Mehrere Bedingungen gleichzeitig prüfen möchtest? (z.B. "Alter zwischen 18 und 65")
- Das Gegenteil einer Bedingung prüfen möchtest? (z.B. "Nicht zugelassen")
- Bedingungen innerhalb von anderen Bedingungen prüfen möchtest? (z.B. "Wenn Alter > 18, dann prüfe ob Führerschein vorhanden ist")

**Wie kann man Bedingungen kombinieren und verschachteln?**

---

## 📚 Theorie

### Logische Operatoren

Mit logischen Operatoren kannst du mehrere Bedingungen kombinieren:

| Operator | Name            | Bedeutung                                | Beispiel                      |
|----------|-----------------|------------------------------------------|-------------------------------|
| `and`    | Logisches UND   | Beide Bedingungen müssen wahr sein       | `alter >= 18 and alter <= 65` |
| `or`     | Logisches ODER  | Mindestens eine Bedingung muss wahr sein | `note == 1 or note == 2`      |
| `not`    | Logisches NICHT | Kehrt den Wahrheitswert um               | `not (alter < 18)`            |

### Operator-Priorität

Die logischen Operatoren haben folgende Priorität (höchste zuerst):

1. `not`
2. `and`
3. `or`

Du kannst Klammern verwenden, um die Priorität explizit festzulegen:

```python
# Ohne Klammern: and wird vor or ausgewertet
if alter > 18 or alter < 13 and erwachsener:
    # Wird interpretiert als: alter > 18 or (alter < 13 and erwachsener)

# Mit Klammern: Explizite Priorität
if (alter > 18 or alter < 13) and erwachsener:
    # Wird so ausgewertet wie geschrieben
```

### Verschachtelte Bedingungen

Du kannst `if`-Anweisungen innerhalb von anderen `if`-Anweisungen schreiben:

```python
alter = 25
fuehrerschein = True

if alter >= 18:
    print("Du bist volljährig")
    if fuehrerschein:
        print("Und du darfst Auto fahren")
    else:
        print("Aber du hast keinen Führerschein")
else:
    print("Du bist noch nicht volljährig")
```

### Wahrheitswerttabellen

#### AND (`and`)

| A      | B      | A and B |
|--------|--------|---------|
| Wahr   | Wahr   | Wahr    |
| Wahr   | Falsch | Falsch  |
| Falsch | Wahr   | Falsch  |
| Falsch | Falsch | Falsch  |

#### OR (`or`)

| A      | B      | A or B |
|--------|--------|--------|
| Wahr   | Wahr   | Wahr   |
| Wahr   | Falsch | Wahr   |
| Falsch | Wahr   | Wahr   |
| Falsch | Falsch | Falsch |

#### NOT (`not`)

| A      | not A  |
|--------|--------|
| Wahr   | Falsch |
| Falsch | Wahr   |

---

## 📋 Zusammenfassung

| Operator | Wirkung                        | Beispiel            |
|----------|--------------------------------|---------------------|
| `and`    | Beide müssen wahr sein         | `x > 0 and x < 100` |
| `or`     | Mindestens eine muss wahr sein | `x == 1 or x == 2`  |
| `not`    | Kehrt den Wert um              | `not (x == 5)`      |

## ✅ Lösung

Hier ist ein Programm, das prüft, ob eine Person für einen Job qualifiziert ist:

```python
# Eingaben
alter = int(input("Wie alt bist du? "))
has_abschluss = input("Hast du einen Abschluss? (ja/nein): ") == "ja"
has_erfahrung = input("Hast du Berufserfahrung? (ja/nein): ") == "ja"

# Prüfung
if alter >= 18:
    if has_abschluss or has_erfahrung:
        print("Du bist für den Job qualifiziert!")
    else:
        print("Du brauchst einen Abschluss oder Berufserfahrung.")
else:
    print("Du musst mindestens 18 Jahre alt sein.")
```
