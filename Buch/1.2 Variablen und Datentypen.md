# 1.2 Variablen und Datentypen

Wie der Computer sich Dinge merken kann

---

## 🎯 Problem

In der letzten Lektion hast du gelernt, wie der Computer mit dir sprechen kann. Aber was ist, wenn du eine Eingabe mehr als einmal verwenden möchtest?

Stell dir vor, du fragst nach einem Namen und möchtest ihn dann mehrmals in verschiedenen Sätzen ausgeben:

```text
Wie heißt du? Anna
Hallo Anna!
Dein Name hat 4 Buchstaben, Anna.
```

**Wie kann der Computer sich den Namen "merken", um ihn später wieder zu verwenden?**

---

## 📚 Theorie

### Was ist eine Variable?

Eine Variable ist wie ein **Behälter** im Computer, in dem du Werte speichern kannst. Du kannst dir eine Variable wie eine beschriftete Schachtel vorstellen:

```python
# So speicherst du einen Wert in einer Variablen:
name = "Anna"
```

Der Name der Variablen (hier: `name`) ist das Etikett auf der Schachtel. Das `=` bedeutet "speichere den Wert auf der rechten Seite in der Variablen auf der linken Seite".

### Wichtige Datentypen

| Datentyp                | Beispiel            | Beschreibung                          |
|-------------------------|---------------------|---------------------------------------|
| **Text (String)**       | `"Hallo"`, `"Anna"` | Alles, was in Anführungszeichen steht |
| **Ganzzahl (Integer)**  | `5`, `42`, `-3`     | Ganze Zahlen ohne Komma               |
| **Dezimalzahl (Float)** | `3.14`, `0.5`       | Zahlen mit Komma/Nachkommastellen     |

### Variablen verwenden

Sobald du einen Wert in einer Variablen gespeichert hast, kannst du ihn immer wieder verwenden:

```python
name = "Anna"
print("Hallo, " + name)        # Gibt "Hallo, Anna" aus
print("Dein Name ist: " + name) # Gibt "Dein Name ist: Anna" aus
```

### Wichtige Regeln für Variablennamen

- Variablennamen dürfen **keine Leerzeichen** enthalten
- Verwende stattdessen **Unterstriche**: `alter_in_jahren` ✅ statt `alter in jahren` ❌
- Variablennamen dürfen **keine Sonderzeichen** wie `!`, `?`, `ä`, `ö`, `ü` enthalten
- Variablennamen sollten **beschreibend** sein: `name` ✅ statt `x` ❌

---

## 📋 Zusammenfassung

| Konzept  | Beispiel        | Beschreibung                           |
|----------|-----------------|----------------------------------------|
| Variable | `name = "Anna"` | Speichert einen Wert unter einem Namen |
| String   | `"Hallo"`       | Text in Anführungszeichen              |
| Integer  | `42`            | Ganze Zahl                             |
| Float    | `3.14`          | Dezimalzahl                            |

---

## ✅ Lösung

Hier ist ein Programm, das sich den Namen merkt und ihn mehrfach verwendet:

```python
# Frage nach dem Namen und speichere ihn
name = input("Wie heißt du? ")

# Verwende die Variable mehrmals
print("Hallo, " + name + "!")
print("Dein Name hat " + str(len(name)) + " Buchstaben.")
print("Schön, dich kennenzulernen, " + name + "!")
```

*(Hinweis: `len()` ist eine Funktion, die die Länge eines Textes zurückgibt. `str()` wandelt eine Zahl in Text um, damit wir sie mit + verbinden können.)*
