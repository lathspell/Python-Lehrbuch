# 1.3 Grundlegende Operatoren

Rechnen und vergleichen mit dem Computer

---

## 🎯 Problem

Du hast gelernt, wie man Variablen verwendet. Aber was, wenn du mit den Werten in den Variablen **rechnen** möchtest?

Stell dir vor:

- Du möchtest das Alter von zwei Personen addieren
- Du möchtest wissen, wie viele Jahre bis zu deinem 18. Geburtstag noch fehlen
- Du möchtest den Durchschnitt von drei Zahlen berechnen

**Wie kann der Computer Mathematik für dich erledigen?**

---

## 📚 Theorie

### Arithmetische Operatoren

Mit Operatoren kannst du mathematische Berechnungen durchführen:

| Operator | Beispiel  | Beschreibung                       |
|----------|-----------|------------------------------------|
| `+`      | `5 + 3`   | Addition                           |
| `-`      | `10 - 4`  | Subtraktion                        |
| `*`      | `3 * 7`   | Multiplikation                     |
| `/`      | `10 / 2`  | Division (ergibt eine Dezimalzahl) |
| `//`     | `10 // 3` | Ganzzahl-Division (ganzer Teil)    |
| `%`      | `10 % 3`  | Modulo (Rest der Division)         |
| `**`     | `2 ** 3`  | Potenz (2³ = 8)                    |

### Beispiele

```python
# Addition
summe = 5 + 3  # Ergebnis: 8

# Subtraktion
differenz = 10 - 4  # Ergebnis: 6

# Multiplikation
produkt = 3 * 7  # Ergebnis: 21

# Division
quotient = 10 / 2  # Ergebnis: 5.0 (Dezimalzahl!)

# Ganzzahl-Division
anzahl = 10 // 3  # Ergebnis: 3 (nur der ganze Teil)

# Modulo (Rest)
rest = 10 % 3  # Ergebnis: 1 (Rest von 10 ÷ 3)

# Potenz
potenz = 2 ** 3  # Ergebnis: 8 (2³)
```

### Operatoren mit Variablen

Natürlich kannst du auch mit Variablen rechnen:

```python
a = 5
b = 3
summe = a + b  # Ergebnis: 8

# Oder direkt in print()
print(a + b)  # Gibt 8 aus
```

## 📋 Zusammenfassung

| Operator | Name              | Beispiel  | Ergebnis |
|----------|-------------------|-----------|----------|
| `+`      | Addition          | `5 + 3`   | `8`      |
| `-`      | Subtraktion       | `10 - 4`  | `6`      |
| `*`      | Multiplikation    | `3 * 7`   | `21`     |
| `/`      | Division          | `10 / 2`  | `5.0`    |
| `//`     | Ganzzahl-Division | `10 // 3` | `3`      |
| `%`      | Modulo            | `10 % 3`  | `1`      |
| `**`     | Potenz            | `2 ** 3`  | `8`      |

---

## ✅ Lösung

Hier ist ein Programm, das verschiedene Rechenoperationen durchführt:

```python
# Lese zwei Zahlen ein
zahl1 = int(input("Erste Zahl: "))
zahl2 = int(input("Zweite Zahl: "))

# Führe verschiedene Berechnungen durch
print("Summe: " + str(zahl1 + zahl2))
print("Differenz: " + str(zahl1 - zahl2))
print("Produkt: " + str(zahl1 * zahl2))
print("Division: " + str(zahl1 / zahl2))
print("Ganzzahl-Division: " + str(zahl1 // zahl2))
print("Rest: " + str(zahl1 % zahl2))
```
