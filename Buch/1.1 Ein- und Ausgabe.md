# 1.1 Ein- und Ausgabe

Wie der Computer mit dir sprechen kann

---

## 🎯 Problem

Stell dir vor, du möchtest, dass der Computer dich nach deinem Namen fragt und dann "Hallo [Dein Name]" sagt. Oder du möchtest zwei Zahlen eingeben und der Computer soll dir die Summe anzeigen.

**Wie kann der Computer mit dir kommunizieren?**

---

## 📚 Theorie

Damit ein Programm mit dir "sprechen" kann, braucht es zwei einfache Befehle:

### Ausgabe: `print()`

Die `print()`-Funktion zeigt Text oder Ergebnisse auf dem Bildschirm an.

```python
print("Hallo Welt")  # Schreibt "Hallo Welt" auf den Bildschirm
```

### Eingabe: `input()`

Die `input()`-Funktion wartet, bis du etwas eingibst, und gibt dann deine Eingabe als Text zurück.

```python
name = input("Wie heißt du? ")  # Fragt nach deinem Namen
```

---

## 📋 Zusammenfassung

| Befehl         | Beispiel                 | Beschreibung                                |
|----------------|--------------------------|---------------------------------------------|
| `print(texte)` | `print("Hallo")`         | Zeigt Text auf dem Bildschirm an            |
| `input(frage)` | `name = input("Name? ")` | Stellt eine Frage und speichert die Antwort |

---

## ✅ Lösung

Hier ist ein einfaches Programm, das den Computer mit dir sprechen lässt:

```python
# Frag nach dem Namen
name = input("Wie heißt du? ")

# Begrüße die Person
print("Hallo, " + name + "! Schön, dich kennenzulernen.")
```

Wenn du dieses Programm ausführst, passiert Folgendes:

1. Der Computer fragt: "Wie heißt du?"
2. Du gibst deinen Namen ein (z.B. "Anna")
3. Der Computer antwortet: "Hallo, Anna! Schön, dich kennenzulernen."
