# 2.1 Bedingungen

Wie der Computer Entscheidungen trifft

---

## 🎯 Problem

Stell dir vor, du möchtest ein Programm schreiben, das:

- Unterschiedliche Nachrichten ausgibt, je nachdem ob eine Zahl positiv, negativ oder null ist
- Nur bestimmte Benutzer Zugang gewährt
- Verschiedene Aktionen durchführt, abhängig von der Tageszeit

**Wie kann der Computer "Entscheidungen" treffen und unterschiedlichen Code ausführen?**

---

## 📚 Theorie

### Die `if`-Anweisung

Mit `if` (deutsch: "wenn") kannst du prüfen, ob eine Bedingung wahr ist:

```python
if Bedingung:
    # Dieser Code wird ausgeführt, wenn die Bedingung wahr ist
    print("Die Bedingung ist wahr!")
```

### Vergleichsoperatoren für Bedingungen

| Operator | Bedeutung           | Beispiel           |
|----------|---------------------|--------------------|
| `==`     | gleich              | `alter == 18`      |
| `!=`     | ungleich            | `name != "Anna"`   |
| `<`      | kleiner als         | `zahl < 10`        |
| `>`      | größer als          | `zahl > 5`         |
| `<=`     | kleiner oder gleich | `alter <= 18`      |
| `>=`     | größer oder gleich  | `temperatur >= 20` |

### `if-elif-else` Kette

Für mehrere Möglichkeiten kannst du `elif` (Else-If) und `else` verwenden:

```python
alter = 15

if alter < 13:
    print("Kind")
elif alter < 20:
    print("Jugendlicher")
else:
    print("Erwachsener")
```

- `if`: Erste Prüfung
- `elif`: Weitere Prüfungen (können beliebig viele sein)
- `else`: Wird ausgeführt, wenn keine der vorherigen Bedingungen wahr war

---

## 📋 Zusammenfassung

| Schlüsselwort | Bedeutung  | Verwendung                         |
|---------------|------------|------------------------------------|
| `if`          | Wenn       | Erste Bedingung prüfen             |
| `elif`        | Sonst wenn | Weitere Bedingung prüfen           |
| `else`        | Sonst      | Standardfall, wenn nichts zutrifft |

---

## ✅ Lösung

Hier ist ein Programm, das eine Note bewertet:

```python
note = int(input("Wie war deine Note? "))

if note == 1:
    print("Sehr gut!")
elif note == 2:
    print("Gut!")
elif note == 3:
    print("Befriedigend!")
elif note == 4:
    print("Ausreichend!")
elif note == 5:
    print("Mangelhaft!")
else:
    print("Ungültige Note!")
```
