# 2.2 Schleifen

Wie der Computer Dinge wiederholt

---

## 🎯 Problem

Stell dir vor, du möchtest:

- Eine Zahl von 1 bis 10 ausgeben
- Alle Zahlen von 1 bis 100 addieren
- Ein Wort mehrmals hintereinander ausgeben
- Eine Eingabe so lange wiederholen, bis der Benutzer "Stopp" eingibt

**Wie kann der Computer dieselbe Aktion mehrmals ausführen, ohne den Code immer wieder zu schreiben?**

---

## 📚 Theorie

### Die `while`-Schleife

Eine `while`-Schleife (deutsch: "solange") führt Code so lange aus, wie eine Bedingung wahr ist:

```python
while Bedingung:
    # Dieser Code wird immer wieder ausgeführt,
    # solange die Bedingung wahr ist
    print("Wiederholung!")
```

**Achtung:** Wenn die Bedingung nie falsch wird, läuft die Schleife **unendlich**!

### Die `for`-Schleife

Eine `for`-Schleife (deutsch: "für") wiederholt Code für jedes Element in einer Sequenz:

```python
for Variable in Sequenz:
    # Dieser Code wird für jedes Element ausgeführt
    print(Variable)
```

Beispiele für Sequenzen:

- `range(5)`: Die Zahlen 0, 1, 2, 3, 4
- `range(1, 6)`: Die Zahlen 1, 2, 3, 4, 5
- `"Hallo"`: Die Buchstaben "H", "a", "l", "l", "o"

### `range()` Funktion

| Aufruf            | Ergebnis                       |
|-------------------|--------------------------------|
| `range(5)`        | 0, 1, 2, 3, 4                  |
| `range(1, 5)`     | 1, 2, 3, 4                     |
| `range(1, 10, 2)` | 1, 3, 5, 7, 9 (Schrittweite 2) |

### Schleifen kontrollieren

- **`break`**: Beendet die Schleife sofort
- **`continue`**: Überspringt zum nächsten Durchlauf

---

## 📋 Zusammenfassung

| Schleife | Verwendung                           | Beispiel             |
|----------|--------------------------------------|----------------------|
| `while`  | Wiederholung basierend auf Bedingung | `while x < 10:`      |
| `for`    | Wiederholung für jedes Element       | `for i in range(5):` |

---

## ✅ Lösung

Hier sind Beispiele für beide Schleifenarten:

```python
# while-Schleife: Zähle von 1 bis 5
zahl = 1
while zahl <= 5:
    print(zahl)
    zahl = zahl + 1

# for-Schleife: Gib alle Buchstaben aus
for buchstabe in "Hallo":
    print(buchstabe)
```
